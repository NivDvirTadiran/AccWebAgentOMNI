The Telephony Window

The Telephoneywindow contains a grid of speed-dial and function buttons that can be assigned various tasks, such as answering contacts or starting applications.

By assigning properties to each button, you can define its function and related information.

 

 

Figure 6: The Telephony Window

 

Ø  To access the Telephony Window:

Click the Windows button in the Agent Toolbar.

 

In the Telephony window, the agent can perform the following actions:

 

To define the properties of a button:

1.         Left-click on the Settings icon of a button.

2.         Select the required category (Telephony, or ACD) and choose the function that you want to assign to this button.

3.         If the selected function requires you to define parameters (for example, a number to dial), the Setup button is enabled. Click this button to open the relevant dialog box and define the required parameters.

4.         Enter the caption that you want to appear on the button and click Save.

To edit the properties of a button:

1.         Left-click the Settings icon of the button .

2.         Edit the properties and click Save.

To clear the properties of a button:

Left-click the Settings icon of the button and click Remove.

To leave the editing process for a button without Saving:

Click Cancel.

To set the mode for all buttons:

Click on of the Mode radio buttons to set the mode for all actions of buttons in the button grid:

Call if the button has a phone number defined, a call will be
initiated.

Transfer if a button has an active call on it, clicking the button
will transfer the call.

Start Consultation if a button has an active call on it, clicking
the button will start the process of establishing a Consultation.
