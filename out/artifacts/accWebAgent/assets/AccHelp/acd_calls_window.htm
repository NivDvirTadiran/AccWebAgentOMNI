The ACD Calls Window

The ACD Calls window displays information about ACD calls that are currently waiting in one of the agents currently logged on queues. The agent can double-click any contact to answer it directly from this window. Information in the Queued Contacts window is displayed in column format. For a list of all available columns, see Available Fields in Agent Windows

 

 

Figure 4: The ACD Calls Window.

 

Ø  To access the Queued Contacts Window:

Click the Windows button in the Agent Toolbar.

 

In the ACD Calls window, the agent can perform the following actions:

 

To answer a contact in the queue:

Double-click the contact to answer.

 

Note: If another agent has already answered the contact, it will not be routed to your destination.

 

To sort the records according to one of the columns:

Click a column heading.

The records are re-ordered according to the selected heading in ascending order. Click the column heading again to arrange them in descending order.

To add a column to the Contact Queue window:

1.    Left-click any column heading and click Insert Column on the shortcut menu. The Insert Column Dialog Box opens.

 

2.    Select the column that you want to insert, define the header, name to appear at the top of the column, alignment and width of the column.

To remove a column from the Contact Queue window:

Left-click the column you want to remove and click Delete Column on the shortcut menu.
